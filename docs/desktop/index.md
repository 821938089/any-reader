## 开发

```sh
# 克隆项目
git clone git@github.com:aooiuu/any-reader.git

# 进入项目根目录
cd any-reader

# 安装依赖
pnpm i

# 编译 解析库、工具库、前端模板
pnpm run build

# 启动
pnpm run electron:server

# 打包
pnpm run electron:build
```
